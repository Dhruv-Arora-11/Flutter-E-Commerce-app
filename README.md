Nike Shoe Store - Flutter E-commerce App
Overview
Welcome to the Nike Shoe Store, an e-commerce mobile app built using Flutter. This app offers a seamless and engaging shopping experience for users looking to browse and purchase Nike shoes.

Features
Modern UI/UX:

Dark-themed design for a sleek look.
Intuitive navigation with a custom bottom navigation bar and a sidebar drawer.
Product Listings:

Detailed product pages with high-quality images and descriptions.
Hot Picks section showcasing popular items.
Search Functionality:

Efficient search bar for quick product discovery.
Shopping Cart:

User-friendly cart to manage selected items.
Easy navigation between home and cart pages.
State Management:

Reactive UI with state management for smooth user interactions.
